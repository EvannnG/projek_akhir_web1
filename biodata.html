Testimonial & Refleksi Kelas

Selama mengikuti mata kuliah Pengembangan Web ini, saya belajar bahwa membangun sebuah website bukan hanya soal membuat tampilan yang menarik, tetapi juga bagaimana mengatur struktur, menjaga konsistensi desain, dan memastikan pengalaman pengguna terasa nyaman di berbagai perangkat.

Bagian yang paling saya nikmati adalah ketika mulai menggunakan Tailwind CSS. Di awal saya sedikit bingung dengan banyaknya utility class, tetapi setelah mencoba lebih jauh, saya merasa proses styling jadi jauh lebih cepat dan rapi. Tantangan terbesar bagi saya adalah mengatur layout responsif agar tetap mirip dengan desain referensi, terutama saat menyesuaikan tampilan untuk layar kecil dan besar.

Proyek akhir ini membantu saya melihat bahwa skill front-end bukan hanya tugas kuliah, tetapi bisa menjadi bagian dari portofolio yang menunjukkan kemampuan saya sebagai calon pengembang web. Saya jadi lebih percaya diri untuk mengerjakan proyek website lainnya ke depannya.
